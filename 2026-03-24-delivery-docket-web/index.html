Changes from Current Delivery Docket List

  • Red "Needs Processing" banner: Shows count of dockets captured on mobile that are missing supplier, PO, or cost centre. Clickable to filter. "Process Now" button to start working through them.
  • Simplified tabs: All, Complete, Needs Processing (with red badge count) — replaces "Complete" and "Incomplete" with clearer language.
  • Visual differentiation: Incomplete rows have a pink background. Missing fields shown as red "✕ Unassigned" tags. Assigned fields shown as green "✓" tags. Immediately obvious what needs attention.
  • "Process →" link: Each incomplete row has a direct action link to open and process that docket.
  • Unknown supplier warning: Orange ! badge on the image thumbnail when OCR couldn't identify the supplier.
  • Removed: Task/Cost Centre combined into just "Cost Centre". Delivery Docket Status and Bills columns removed for clarity — status is now the processing status (Incomplete/To be paid/Paid).
  • Why: The current view treats incomplete and complete dockets the same visually. Photo-captured dockets from mobile need immediate attention — this redesign makes them impossible to miss.
