# Submission Dashboard

## What this is

Analytics dashboard for Custom Form submissions — shows total submission count, status breakdown, and multi-dimensional breakdowns by submitter, form, source, equipment, and project.

## Screens

1. **Submission Dashboard** — filters, total count hero, status cards, six breakdown panels

## Key elements

- **Filters bar** — Date Range, Form, Respondant, More Filters
- **Hero banner** — total submissions (284) in large type on dark navy background
- **By Status** — Unknown Status vs Needs Action, with red left-border highlight on action-required card
- **Submitted By** — horizontal bar chart per user with coloured avatar initials, count, percentage (scrollable for 16 submitters)
- **Assign To Display** — submissions grouped by assignment category (Equipment, Employee, User, Project, Unassigned)
- **By Form** — per-form submission counts with colour-coded bars (scrollable for 29 forms)
- **Submission Source** — breakdown by entry point (Project, Timesheet, Equipment, Normal)
- **By Equipment** — submissions linked to specific equipment units
- **By Project** — per-project submission counts
