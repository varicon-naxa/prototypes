Changes from Current Add Project Form

  • Job Number: Pre-filled with next in sequence (VAR-001). Editable in a future project settings step.
  • Removed: Contract Number, Purchase Order Number, Segment, Status (always Active for new projects).
  • Tracking Category (Xero): Mandatory. Greyed-out fields below until selected. Sync with Xero button. Option to create new tracking category at bottom of list.
  • Start Date: Defaults to today. End Date shows calculated project duration underneath.
  • Client Details: Now includes Client Contact dropdown (appears after selecting client). "+ Add New Contact" option at bottom of list.
  • Payment Act: Auto-set based on the State selected in Job Address. Shows green confirmation.
  • Resource Cost Sheet: Hidden (handled during onboarding setup).
  • Assign Cost Centre: Moved below Estimation Sheet. Includes explanation that it's only required if not uploading an estimation sheet (which contains cost centres).
