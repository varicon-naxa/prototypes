Product Spec — Toolbar Cleanup

  • Default view (collapsed): Date Range, Filters (with active count badge), Search, Need Help?, and a "More" button
  • Expanded view: Clicking "More" reveals a second row with: Add Project, Import Projects, Download Template
  • Why: The current toolbar is crowded for everyday use. Most users only need date range, filters, search, and help day-to-day. Project creation and import actions are one click away but not cluttering the default view.
  • Interaction: "More" button toggles — click to expand, click again to collapse. State resets on page load (always starts collapsed).
